## 🎯 Overview

**gourl** is a lightweight yet powerful CLI-based file uploader that allows you to upload files directly from your terminal to **temp.sh** and instantly get a shareable URL along with a QR code.

Designed for speed, simplicity, and real-world usage.

---

## ⚡ Features

- 📂 Multi-file upload support  
- 📊 Real-time upload progress bar  
- 🔗 Instant URL generation  
- 🔳 QR code generation (terminal-based)  
- ⚙️ Clean & minimal output  
- 🌍 Cross-platform support  
  - Termux  
  - Linux  
  - Windows  
- 🧠 Lightweight & fast

## ⚡ Installation

## Kali Linux & Termux

```bash
git clone https://github.com/MRXVOID/gourl.git
cd gourl
python3 setup.py
```
## Windows [CMD]

Download Zip 👇

```bash
https://github.com/MRXVOID/gourl/archive/refs/heads/main.zip
```

## 🚀 Usage

### 🔹 Basic Upload
```bash
gourl -f file.txt
```

## 🔹 URL + QR Code
```bash
gourl -f file.txt -u -qr
```

## 🔹 Multiple Files
```bash
gourl -f file1.txt file2.jpg file3.zip
```

## 🔹 For Help
```bash
gourl --help
```
